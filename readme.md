# 🎬 NodeJS API developed using ExpressJS and Typescript

### The REST API for a sample blog database

## ⬇ Install Dependencies

> **npm install**

## ⚒️ Build - Transpiles the typescript into Javascript code into dist folder in root directory

> **npm run build**

## ⚒️ Run app

> **npm run start**

## ⚒️ Start dev server

> **npm run dev**
